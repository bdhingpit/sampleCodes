SHORT DESCRIPTION/GOAL
-Classify whether a movie review is positive of negative

DATASET
-Built-in in keras library. Contains 25,000 movie reviews (15,000 training set and 10,000 validation set). Used 10,000 most common words used in reviewing movies.

MODEL USED
-Artificial neural network (ANN). Two 16-node hidden layer with ReLU activation function. Output layer has 1 node with a sigmoid activation function.

RESULT
-For model 1, 99.92% accurate on training set and 86.85% accurate on validation set.
